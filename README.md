## How to build the Book

Make sure you have `pdflatex` installed (comes with TeX Live, MikTeX, etc.)

To compile:

```bash
make pdf
```

To clean all build artifacts:
```bash
make clean
```

Final output PDF will be named:
bitcoin_learning.pdf
